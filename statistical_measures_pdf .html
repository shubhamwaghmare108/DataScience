1. Measures of Central Tendency

1.1 Mean (Average)

The sum of all values divided by the number of values.

Mean = Œ£x / n

Advantages:

  • Uses all data points
  • Easy to calculate and understand
  • Good for normally distributed data

Disadvantages:

  • Affected by outliers
  • May not represent skewed data well
Example: Data: 10, 15, 20, 25, 30 ‚Üí Mean = 100/5 = 20

1.2 Median

The middle value when data is arranged in order.

Advantages:

  • Not affected by outliers
  • Good for skewed distributions
  • Easy to understand
Example: Data: 10, 15, 20, 25, 30 ‚Üí Median = 20

1.3 Mode

The value that appears most frequently in the dataset.

Example: Data: 10, 15, 15, 20, 30 ‚Üí Mode = 15
Measure Best Used When Outlier Affected
Mean Normally distributed Yes
Median Skewed/outliers No
Mode Categorical data No

4. Measures of Position

4.1 Quartiles

Divide data into 4 equal parts (25% each).

  • Q1: 25th percentile
  • Q2: 50th percentile (Median)
  • Q3: 75th percentile
Example:
Data: 10, 15, 20, 25, 30, 35, 40, 45, 50
Q1=20, Q2=30, Q3=40, IQR=20

4.2 Deciles

Divide data into 10 equal parts (10% each).

D1 = 10th percentile, D2 = 20th percentile, ..., D9 = 90th percentile

4.3 Percentiles

Divide data into 100 equal parts (1% each).

Position = (P / 100) √ó (n + 1)

Common Percentiles:

  • P10 = 10th percentile
  • P25 = 25th percentile (Q1)
  • P50 = 50th percentile (Median)
  • P75 = 75th percentile (Q3)
  • P90 = 90th percentile

6. Quick Reference Guide

When to Use Each Measure

Central Tendency:

  • Mean: Normally distributed, outlier-free
  • Median: Skewed or outliers present
  • Mode: Categorical or most common value

Dispersion:

  • Range: Quick overview
  • SD: Normal data without outliers
  • IQR: Skewed data or outliers
  • CV: Comparing different scales

Shape:

  • Skewness: Direction bias
  • Kurtosis: Outlier risk, tail behavior

Position:

  • Quartiles: Distribution overview
  • Deciles: Rankings, income analysis
  • Percentiles: Test scores, comparisons
  • IQR: Outlier detection
