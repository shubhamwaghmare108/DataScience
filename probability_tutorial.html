1. FUNDAMENTALS OF PROBABILITY

What is Probability?

Probability measures the likelihood that an event will occur. It ranges from 0 (impossible) to 1 (certain).

Basic Definitions

Sample Space (S)

Set of all possible outcomes.

Example: Coin flip ‚Üí S = {Heads, Tails}

Event (E)

A specific outcome or group of outcomes.

Probability Formula

P(E) = Number of Favorable Outcomes / Total Number of Possible Outcomes

Probability Range

0 ‚â§ P(E) ‚â§ 1
P(E) = 0 ‚Üí Impossible
P(E) = 0.5 ‚Üí Equally likely
P(E) = 1 ‚Üí Certain

Types of Probability

  • Classical: Equally likely outcomes (dice, coins)
  • Empirical: Based on observed frequency
  • Subjective: Based on personal judgment

Basic Examples

Example 1: P(Heads) = 1/2 = 0.5 = 50%
Example 2: P(Rolling 3 on die) = 1/6 ‚âà 16.7%
Example 3: P(Drawing Ace from deck) = 4/52 ‚âà 7.7%

2. BASIC PROBABILITY RULES

Complement Rule

P(E') = 1 - P(E)

Probability of event NOT occurring.

Example: P(Not getting 6) = 1 - 1/6 = 5/6

Addition Rule

Mutually Exclusive Events (no overlap)

P(A or B) = P(A) + P(B)
Example: P(Rolling 1 or 2) = 1/6 + 1/6 = 1/3

Non-Mutually Exclusive Events (with overlap)

P(A or B) = P(A) + P(B) - P(A and B)
Example: P(Card is Red or Ace)
= 26/52 + 4/52 - 2/52 = 28/52 = 7/13

Multiplication Rule

Independent Events

P(A and B) = P(A) √ó P(B)
Example: P(Heads AND 6 on die)
= 1/2 √ó 1/6 = 1/12

Dependent Events

P(A and B) = P(A) √ó P(B|A)
Example: P(Two Aces without replacement)
= 4/52 √ó 3/51 ‚âà 0.0045

3. CONDITIONAL PROBABILITY

Definition

P(B|A) = P(A and B) / P(A)

Read as: "Probability of B given A"
Example: P(King | Face Card) = 4/12 = 1/3

Bayes' Theorem

P(A|B) = P(B|A) √ó P(A) / P(B)

Medical Test Example

Scenario:
‚Ä¢ Disease is rare: P(Disease) = 0.01
‚Ä¢ Test accuracy: P(Positive|Disease) = 0.99
‚Ä¢ False positive rate: P(Positive|No Disease) = 0.05

Question: If you test positive, what's probability you have disease?

Answer: P(Disease|Positive) ‚âà 16.7%
(Much lower than 99% because disease is rare!)

Key Concepts

  • Independence: Events where outcome of one doesn't affect the other
  • Mutually Exclusive: Events that cannot both occur
  • Exhaustive: Events together cover all possibilities
  • Law of Large Numbers: As trials increase, experimental probability approaches theoretical probability

Problem-Solving Tips

  • Check if events are independent or dependent
  • "Or" suggests addition, "and" suggests multiplication
  • Use complement when easier to find P(not E)
  • Watch for with/without replacement
  • Use Bayes' Theorem for updating beliefs with new information
  • Verify all probabilities sum to 1
